# 🏋️ Movo — App de Hábitos

App de seguimiento de hábitos con gamificación RPG.

**TFG 2DAM · Florida Universitària · 2025-2026**

## 🛠️ Tecnologías

- 📱 Frontend: React Native + Expo
- ⚙️ Backend: Java + Spring Boot
- 🗄️ Base de datos: MySQL (Railway)
- 🔐 Autenticación: JWT
- 🎨 Diseño: Figma

## 🚀 Cómo ejecutar

### Backend
```bash
cd backend
./mvnw spring-boot:run
```

### Frontend
```bash
cd frontend
npx expo start
```
